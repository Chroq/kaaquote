# Kaaquote

### Experimental project (educational)

This REST API (based on Rocket, a Rust web framework) gets quotes of the french TV show Kaamelott to feed a Slack command.  

## Installation

```bash
make install
```

## Usage

```bash
make start
```

## Test

```bash
make test
```
